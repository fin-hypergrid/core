The Scrollbars tab

The default scrollbar stylesheet

Before proceeding, it is worth studying the default scrollbar stylesheet, finbars.css, which is instructive for whichever approach you decide to take.

From finbars.css, we can see that CSS styles for scrollbars can control the following (defaults in bold):

  • Scrollbar position
    • over content vs. adjacent to content
    • vertical scrollbar: right of content vs. left
    • horizontal scrollbar: below content vs. above
  • Gutters
    • size1 (11px)
    • margins relative to grid rect (3px all around)
    • background color (none, i.e., transparent)
    • border color and thickness (none)
  • Thumbs
    • size1 (7px)
    • margins relative to gutter rect (2px all around)
    • position within gutter (centered*)
    • shape: square vs. rounded
    • background color (lightgrey, or #d3d3d3)
    • border color and thickness (none)
    • shadow2 (black, 1px)
  • Hover styles
    • All of the above can be modified during mouse hover (thumb opacity: 1.0)
    • Transition time for selected styles between hover vs. non-hover (thumb opacity: ½ second)

1Size refers to the width of the vertical scrollbar and its thumb; or to the height of the horizontal scrollbar and its thumb. Important note: Never attempt to set the size of the other axis as this is programmatically determined and set by the code, based on the size of the logical grid within its currently sized viewport.
2The box-shadow style with offsets of 0px can be used as a subtler alternative to a border. You should use either border or shadow, or neither, but not both. If you prefer to use border, you will need to fiddle with margin and/or box-sizing to keep the thumb positioned properly within the gutter.

The Cell Editors tab

CE-1
Edit a cell without a localizer

Before we try cell editors, we’ll remove all localizer bindings. Cell editors can function with or without localizers; we want to try it without localizers first.

  1. Switch to the State tab
  2. Click the button to restore state to the default (no localizers)
Okay, now we’re ready.
  1. Open a cell editor by double-clicking any cell
  2. An editor is superimposed over the cell
  3. Edit the value
  4. Save the edited value by pressing the enter or tab keys
    1. the edited value is saved to the data model
    2. the editor is removed
    3. the grid is re-rendered to display the new value
Or instead of saving the edit, try repeating steps 3 through 5 and then:
  1. Abort the edit by pressing the esc key
    1. the editor is removed
    2. the grid retains the old value

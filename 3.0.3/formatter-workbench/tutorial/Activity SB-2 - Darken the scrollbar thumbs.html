The Scrollbars tab

Styling the scrollbars

For custom scrollbar styling, you have two choices:

  1. Override some styles in the default scrollbars stylesheet
    • Create your own “scrollbars.css” stylesheet with rules that override selected styles, or add the rules to your app’s general stylesheet
    • Be sure to locate your stylesheet below the default stylesheet within the DOM so your rules will override the default rules
    • (Leave finbars’s default stylesheet in place)
  2. Replace the default scrollbars stylesheet
    • Create your own “scrollbars.css” stylesheet with rules that completely describe the scrollbars, or add the rules to your app’s general stylesheet
    • Be sure to include the two “DO NOT TOUCH” styles documented in finbars.css, i.e., position: absolute for both gutters and thumbs
    • Remove or disable finbars’s default stylesheet (in this workbench, just click the “Replace” checkbox in the Scrollbars tab)
    • (In this case your stylesheet location within the DOM is not important)

Darken the scrollbar thumbs

In this activity, we shall take the first of the two approaches discussed above and override the thumb rule’s background-color style established in the default scrollbars stylesheet.

  1. Open the default scrollbars stylesheet, finbars.css, containing the styles we want to override, and place the window off to the side
  2. In the stylesheet, the default thumb background color in the “thumb style” rule is lightgrey (#d3d3d3)
  3. Switch to the Scrollbars tab and click its reset icon, if red (), to reload the empty custom stylesheet, which consists of all the commented selectors from the default stylesheet, except with all the CSS rules emptied (so, so far it’s overriding nothing)
  4. Add a background-color override to the additional “thumb style” rule here (in the editor) with a darker color, such as medium black, so it looks like this: 
    /* thumb style */
div.finbar-vertical > .thumb, div.finbar-horizontal > .thumb {
    background-color: black;
}
    This is a stylesheet. We use CSS syntax here, not JSON or JavaScript. That is, do not use quotes; do not use camelCase.
  5. Click the injectCSS('custom', …) button to inject this text into a style element, which is appended to the DOM just below the default scrollbars stylesheet style element
  6. Thumbs are now darker.

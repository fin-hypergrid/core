The Scrollbars tab

Repositioning the scrollbars off the grid content

The following activity employs a simple strategy for moving the scrollbars off the grid content. Rather than move the scrollbars into “uncharted” screen real estate, this strategy instead shrinks the grid just enough to move it away from the scrollbars, which actually remain where they were. This is accomplished by setting the grid’s bottom and right margins.
Grid margins can also be set as an option to the Hypergrid constructor.
SB-4
Reposition the scrollbars
  1. In the Scrollbars tab, shrink the grid by applying borders: 
    div.hypergrid-container > div:first-child {
    bottom: 13px;
    right: 11px;
    box-shadow: none !important; /* remove workbench style */
}
    The box-shadow setting would not be needed normally; it just undoes the fuzzy shadow around the grid that this workbench applies, and which was interfering visually with the new scrollbar positioning.
  2. Click the injectCSS('custom', …) button to update the custom style element
  3. ALthough the scrollbars have not actually moved, they no longer overlay the grid because we’ve made the grid a little smaller.

Again, we urge you to go back to the first page of this section and take a look at its Code Snippets tab. There you will find examples of gutter borders and of moving the scrollbars to the opposite edges of the grid.

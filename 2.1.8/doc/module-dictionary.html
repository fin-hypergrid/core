(static) decorate(list, indexopt)

Add dictionary entries to an array.

Adds entries for a single element or the entire array.

Each element can be either a string or an object with a name property that is a string.

All converters are run on that string and the resulting unique non-blank "entries" are added as properties to the array with the value of the property being a reference to the element (if it was an object) or a copy of the element (if it was a string), except that

  • Duplicate entries are not added.
  • Blank entries are not added.
  • Entries that begin with a digit are prefixed with with $.
Parameters:
Name Type Attributes Description
list
index number <optional>

List element to add entries for. If omitted, adds entries for all elements.

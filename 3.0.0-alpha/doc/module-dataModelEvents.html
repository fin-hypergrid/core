(static) dispatchDataModelEvent(event)

Hypergrid's data model event handler.

This dispatchEvent function is handed to the data model to issue callbacks to the grid.

This handler:

  1. Checks the event string to make sure it conforms to the expected syntax (must start with fin-hypergrid-data- or fin-hypergrid-schema- and include only lowercase letters and hyphens).
  2. Calls a function with that name in dataModelEventHandlers.
  3. Re-emits the event as a DOM event to the <canvas> element.

The data model's dispatchEvent method is bound to the grid by injectCode. A curried version of this function, bound to the grid instance, is either: Added to the data model via its addListener method, if it has one; or Force-injected into the data model, overriding any native implementation. (A native implementation may exist simply to "catch" calls that might be made before the data model is attached to Hypergrid.)

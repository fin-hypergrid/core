Tutorial pane

The tutorial pane has two tabs.

Tutorial tab

You are looking at the Tutorial tab.
  • The tutorial is split into easily digestible, page-size chunks.
  • Depending on your window size, you may need to scroll down to see the whole page.
  • To move from page to page:
    • Click the ◀ or ► buttons above
    • Drag the slider for random access to any page
    • Press the ← or → keys on your keyboard (blocked when a text field has the focus)

Code Snippets tab

This tab contains code snippets for cutting-and-pasting and only appears when the current tutorial page has such content.

Click the ► button (above) to go to the next page.
